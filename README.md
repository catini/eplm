# EPLM – Embodied Programming Learning Method

Ferramentas digitais de apoio ao **Embodied Programming Learning Method (EPLM)**, um método baseado em Aprendizagem Corporificada para o ensino de conceitos de Programação Orientada a Objetos (POO).

## Sobre o projeto

O EPLM integra atividades físicas, ferramentas tangíveis e sistemas interativos para facilitar a compreensão de conceitos abstratos da POO. Este repositório contém as ferramentas digitais desenvolvidas no âmbito da pesquisa.

## Estrutura

- [`eplmh/`](./eplmh) — Ferramenta de reconhecimento de imagens para o ensino do conceito de **Herança**
- [`eplmp/`](./eplmp) — Ferramenta de reconhecimento de movimento para o ensino do conceito de **Polimorfismo**

## Tecnologias

- HTML5, CSS3, JavaScript
- [TensorFlow.js](https://www.tensorflow.org/js)
- [Teachable Machine](https://teachablemachine.withgoogle.com/)

## Como executar

As ferramentas funcionam diretamente no navegador, sem necessidade de instalação:

1. Faça download do repositório (botão **Code → Download ZIP**) ou clone via Git
2. Extraia o arquivo ZIP em uma pasta de sua preferência
3. Abra o arquivo `index.html` da ferramenta desejada (`eplmh` ou `eplmp`) em um navegador moderno (Chrome, Edge ou Firefox)
4. Conceda permissão de acesso à webcam quando solicitado
